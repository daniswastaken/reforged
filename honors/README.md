# Honors
We believe that every contributions towards this project should be acknowledged and honored properly.

This section is intended to be used for honoring all contributions from everyone.

If you think your contribution hasn't been properly honored, you can try [Opening an Issue](https://github.com/daniswastaken/reforged/issues) on GitHub or sending an email to [hi@handaru.dev](mailto:hi@handaru.dev) and we'll try to get it resolved as quickly as possible.

Every other 3rd party projects or development resources provided by the community used in this project, its license is put at the [license directory](./licenses/) for easier organizing and management.
